# Субтитры (интро)

## Оригинал (Kyrandia / ScummVM)
- Интро описано скриптом последовательностей. Внутри них есть опкоды вывода текста.
- Два режима текста:
  - **Центрированный текст** (письмо): выводится внизу (y≈180 на экране 320x200), по центру.
  - **Реплики**: выводятся по явным координатам (x/y) из данных сцены. Координата y — это нижняя опорная линия.
- Тайминги идут в тиках (1 тик ≈ 1/60 сек). Текст держится до следующей реплики или конца сцены.

## В этом проекте
- Субтитры перенесены 1:1 по таймингам из оригинала и переведены в миллисекунды.
- Тексты лежат отдельно в `src/engine/intro/introText.json`.
- Тайминги/позиции — в `src/engine/intro/subtitles.ts`, подключаются в `src/engine/intro/steps.ts`.
- Рендер идёт через DOM‑оверлей (`src/components/SubtitleOverlay.vue`) поверх canvas.
- Плеер интро отдаёт активные строки через `onSubtitleChange`, чтобы оверлей обновлялся без рисования текста на canvas.

## Примечания
- Цвет задаётся для каждой строки отдельно, чтобы совпадать с оригиналом.
- Оверлей настроен под «пиксельный» рендер. При переходе на bitmap‑шрифт это можно сделать без изменений в canvas.
